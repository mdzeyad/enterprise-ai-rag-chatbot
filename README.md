# 🤖 Enterprise AI Chatbot (RAG System)

## 🚀 Overview

This project is an AI-powered chatbot that uses Retrieval-Augmented Generation (RAG) to answer questions from uploaded documents.

---

## 🧠 Features

* PDF document Q&A system
* Semantic search using embeddings
* Context-aware responses using LLMs
* Real-time chatbot interface

---

## 🛠️ Tech Stack

Python, LangChain,  OpenAI API, Streamlit

---

## ⚙️ How it works

1. Upload document
2. Convert text into embeddings
3. Store in vector database (FAISS)
4. Retrieve relevant context
5. Generate response using LLM

---

## 📌 Use Cases

* Enterprise knowledge assistant
* Customer support automation
* Document search system
